Student Innovation Project

The Hybrid Controller

The Hybrid Controller is a combination of an arcade fightstick used by many fighting game players and an analog joystick from normal controllers found on Xbox and Playstation consoles. The base of the controller contains the 8-button layout found on most fightsticks and is connected to a "nunchuck" peripheral through set of wires. Speaking of the “nunchuck” this built-in peripheral contains both an analog joystick and a D-pad for movement in games, and 2 buttons on the back.


The main innovation of the Hybrid Controller is its built-in “nunchuck” that improves on the original peripheral for the Wii. Unlike the Wii nunchuck, the Hybrid Controller's "nunchuck" has a D-pad so players have the option to use digital inputs for movement in games.
